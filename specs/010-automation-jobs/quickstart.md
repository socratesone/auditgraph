# Quickstart: Automation and Jobs

## Goal

Understand job configuration, manual execution, and output locations.

## Steps

1. Review [specs/010-automation-jobs/spec.md](specs/010-automation-jobs/spec.md) for requirements.
2. Check [specs/010-automation-jobs/data-model.md](specs/010-automation-jobs/data-model.md) for schema rules.
3. Use [specs/010-automation-jobs/contracts/automation-jobs.openapi.yaml](specs/010-automation-jobs/contracts/automation-jobs.openapi.yaml) for CLI output fields.

## Success Check

- Job configuration schema is documented.
- Manual `jobs run` behavior is documented.
- Output paths are deterministic and documented.
